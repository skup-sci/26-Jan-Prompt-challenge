# Product Overview

## Multilingual Mandi Platform

A voice-first multilingual web platform (hackathon MVP) for local vendors in Indian markets to check mandi prices and negotiate across languages using AI.

## Core Purpose

Enable local vendors with varying digital literacy to easily check prices and negotiate in their native language through a simple, voice-driven interface.

## MVP Focus (Hackathon Demo)

This is a simplified MVP focused on core functionality:

- **Voice-First Interface**: Primary interaction through voice commands and speech
- **Multilingual Communication**: Support for major Indian languages (Hindi, English, Tamil, Telugu, Bengali, Marathi, Gujarati, Kannada, Malayalam, Punjabi)
- **AI Price Discovery**: Simple price checking for commodities
- **Negotiation Assistance**: Basic AI-guided negotiation support
- **Mobile-First UI**: Large buttons, minimal text, farmer-friendly design

## What We're NOT Building (MVP Scope)

- Complex authentication systems
- Analytics dashboards
- Enterprise security features
- Large backend systems
- Vendor profiles or trust systems
- Transaction recording/history

## Target Users

Local vendors and traders in Indian mandis with varying digital literacy levels who need simple, voice-driven tools.

## Design Philosophy

- **Hariyali (Green) Theme**: Agricultural, nature-inspired colors
- **Large Touch Targets**: Easy to tap buttons for all users
- **Minimal Text**: Icon-driven, voice-first approach
- **Simple Navigation**: Few screens, clear purpose
